**After (fixed)** — uses a named parameter `:city` with `setParameter`:
```java

        .setParameter("city", "%" + city + "%")
        .getResultList();
```

The `%` wildcards for the `LIKE` clause are included in the parameter value passed to `setParameter`, not in the query string itself. This ensures any malicious input (e.g., `' OR '1'='1`) is treated as a literal string value and never interpreted as JPQL syntax, eliminating the injection vulnerability.
